1. **Repeat Best not updating (major):** The "Brew Again" form on `/brew/best` uses a hardcoded `recommendation_id="best-{{ best.id }}"` which never changes between visits. Purpose: Close the two remaining UAT gaps so the Phase 3 optimization loop is fully functional.

     async def deactivate_bean(request: Request):
