    The function takes a Brewer ORM object and returns an integer 1-5:

    ```python
    def derive_tier(brewer) -> int:
        """Derive UX tier (1-5) from brewer capability flags.

        Tier 1 — Basic: grind, dose, yield only
        Tier 2 — Temperature Control: + temperature
        Tier 3 — Pre-infusion: + preinfusion_time
        Tier 4 — Pressure/Flow Control: + brew_pressure/pressure_profile
        Tier 5 — Full Programmable: + flow_rate, preinfusion_pressure, bloom
        """
        # Tier 5: programmable flow control (Decent DE1, Meticulous)
        if brewer.flow_control_type == "programmable":
            return 5

        # Tier 4: any flow control OR programmable/manual pressure profiling
        if brewer.flow_control_type in ("manual_paddle", "manual_valve"):
            return 4
        if brewer.pressure_control_type in ("manual_profiling", "programmable"):
            return 4

        # Tier 3: timed or adjustable pre-infusion
        if brewer.preinfusion_type in ("timed", "adjustable_pressure", "programmable"):
            return 3

        # Tier 2: PID or profiling temperature control
        if brewer.temp_control_type in ("pid", "profiling"):
            return 2

        # Tier 1: basic (no controllable parameters beyond grind/dose/yield)
        return 1
    ```
