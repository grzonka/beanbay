  **Model change (measurement.py):**
  - Add `flavor_tags = Column(String, nullable=True)` after the intensity column (line 36). This stores JSON-encoded list of tag strings. Use String (not JSON type) for SQLite compatibility. The router will json.dumps on save and json.loads on read.
  - NOTE: The 6 flavor dimension columns (acidity, sweetness, body, bitterness, aroma, intensity) and `notes` column already exist in the model. Do NOT add them again.
  - Since this app uses `Base.metadata.create_all()` at startup (no Alembic migrations directory exists), the new column will be created automatically on fresh databases. For existing databases, add a startup migration helper: in `app/main.py` lifespan, after `create_all()`, add a block that checks if the `flavor_tags` column exists in the `measurements` table using `inspect(engine).get_columns("measurements")` and adds it via `ALTER TABLE measurements ADD COLUMN flavor_tags TEXT` if missing. This is a common pattern for SQLite single-user apps.

  **Record endpoint (brew.py):**
  - Add these optional Form parameters to `record_measurement`:
    - `notes: Optional[str] = Form(None)` 
    - `acidity: Optional[float] = Form(None)`
    - `sweetness: Optional[float] = Form(None)`
    - `body: Optional[float] = Form(None)`
    - `bitterness: Optional[float] = Form(None)`
    - `aroma: Optional[float] = Form(None)`
    - `intensity: Optional[float] = Form(None)`
    - `flavor_tags: Optional[str] = Form(None)` — this is a comma-separated string from the form; convert to JSON list on save
  - When creating the `Measurement` object (around line 176), add all these fields:
    - `notes=notes.strip() if notes else None`
    - For each flavor dimension: only set if value is not None (slider was touched). Clamp to 1-5 range.
    - `flavor_tags=json.dumps([t.strip() for t in flavor_tags.split(",") if t.strip()][:10]) if flavor_tags else None`
  - Add `import json` at top of file
  - Also import `inspect` from `sqlalchemy` in `app/main.py` for the migration helper

  **Startup migration (main.py):**
  - After `Base.metadata.create_all(bind=engine)` (line 27), add:
    ```python
    from sqlalchemy import inspect as sa_inspect, text
    inspector = sa_inspect(engine)
    existing_cols = {c["name"] for c in inspector.get_columns("measurements")}
    if "flavor_tags" not in existing_cols:
        with engine.begin() as conn:
            conn.execute(text("ALTER TABLE measurements ADD COLUMN flavor_tags TEXT"))
    ```

      </div>

            placeholder="Type a flavor (e.g. chocolate, citrus)..."

    </div>
  </div>

  - Tags: `<datalist>` provides suggestions. JS handles adding/removing tags. Hidden input `flavor_tags` holds comma-separated values.
