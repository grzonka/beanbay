**Route change (app/routers/brew.py):**

Update the `brew_index` GET handler (around line 110-123). Instead of redirecting to /beans when no active bean, render the brew/index.html template with a flag indicating no bean is active:

```python
@router.get("", response_class=HTMLResponse)
async def brew_index(request: Request, db: Session = Depends(get_db)):
    """Main brew page — entry point for the optimization loop."""
    bean = _require_active_bean(request, db)
    if not bean:
        return templates.TemplateResponse(
            request,
            "brew/index.html",
            {"no_active_bean": True},
        )

    has_measurements = db.query(Measurement).filter(Measurement.bean_id == bean.id).count() > 0

    return templates.TemplateResponse(
        request,
        "brew/index.html",
        {"active_bean": bean, "has_measurements": has_measurements},
    )
```

**Do NOT change** the other routes (POST /brew/recommend, GET /brew/best, POST /brew/record) — they still redirect to /beans because they genuinely need a bean to function.
