    - self._cache: dict[str, Campaign] = {} (in-memory campaign cache)
    - self._lock = threading.Lock() (BayBE is NOT thread-safe)

    **Campaign path: _campaign_path(self, bean_id: str) -> Path:**
    - Return self._campaigns_dir / f"{bean_id}.json"

    **Get or create: get_or_create_campaign(self, bean_id: str) -> Campaign:**
    - Check cache first
    - If not cached, check disk (path.exists() -> Campaign.from_json(path.read_text()))
    - If not on disk, create fresh campaign and save to disk
    - Return cached campaign
    - This method acquires the lock internally

    **Recommend: async recommend(self, bean_id: str) -> dict:**
    - Use asyncio.to_thread to push blocking BayBE work off the event loop
    - Inside the thread: get_or_create_campaign, then with self._lock: campaign.recommend(batch_size=1)
    - After recommend, save campaign to disk (recommend modifies internal state — marks points as recommended)
    - Round the recommendation to practical precision:
      - grind_setting: round to nearest 0.5
      - temperature: round to nearest 1.0
      - preinfusion_pct: round to nearest 5.0
      - dose_in: round to nearest 0.5
      - target_yield: round to nearest 1.0
      - saturation: leave as-is (categorical)
    - Generate a recommendation_id (uuid4) and include it in the returned dict
    - Return dict with all 6 params + recommendation_id

    - Create a fresh campaign
    - If measurements_df is not empty, add all measurements (only BayBE columns: 6 params + taste)
    - Save to disk and cache
    - Return the new campaign

    - Get campaign from cache, write JSON to disk

import tempfile
